解决BindingException问题

解决org.apache.ibatis.binding.BindingException: Type interface xxx is not known to the MapperRegistry

运行SpringMVC时控制台报错

解决办法

1.检查定义的接口和实体类映射的接口名是否一致



特别注意的是当项目只有一个包的时候可以只写com.zs.dao，但是有多个包，要具体到类名com.zs.dao.StudentDao。

2.配置文件里xml扫描器扫描

在配置文件中我发现使用.xml出现了这个错误，把改成具体我要实现的那个接口，解决了问题。
