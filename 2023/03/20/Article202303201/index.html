ChatGPT 的几种访问途径及开发学习入门参考

ChatGPT 的几种访问途径及开发学习入门参考

1 注册一个 OpenAI 账号

如果你想直接访问官网服务或者获取 API Key 用于开发，首先需要注册一个 OpenAI 账号。注册流程如下：

  1. 找一个可用的网络环境，例如可以使用 VPN、购买一定时间的会员等方式来访问官网并注册账号，免费的节点基本很容易因同 IP 注册太多而失败。
  2. 准备一个邮箱。现在不建议使用 QQ、163 等邮箱，推荐使用Gmail邮箱。
  3. 由于目前 OpenAI 要求国外手机号码进行验证，需要使用接码平台等方式获取美国手机号码来接收验证码。
  4. 前往 OpenAI 官网注册页面ChatGPT 注册/登录，按照流程操作即可。
  5. 注册成功后，可点击右上角个人头像，弹出菜单中可以找到生成 Keys 的选项链接。新账号默认有 5 美元余额，如果是该手机注册的第一个账号则有 18 美元，当前注册的有效期基本都是截止至7月1日。

2 体验 ChatGPT 功能的几种途径

国内访问并体验 ChatGPT 功能存在一定的门槛，主要受限于网络无法访问、账号无法注册等条件。根据你所拥有的条件不同，可以有以下几种途径使用 ChatGPT 的功能。

2.1 无 VPN、无 API KEY：使用免费在线应用

对于大多数人来说，由于无法访问外网，也没有 API KEY，因此无法直接使用 OpenAI 官网提供的服务。但是，在开源社区中有很多基于 OPENAI API 搭建的在线应用网站，可以免费使用。例如，https://github.com/lzwme/chatgpt-sites 就提供了许多免费体验 ChatGPT 的在线网站，其中包括基于该背景的脚本程序，可根据 GitHub 上开源程序的 fork 信息分析并抓取这样的网站地址并汇总，基于 GitHub Action 每天定时执行。目前已汇总网站数量 1024 个，日均新增 100+ 个。如果你有兴趣了解，可以访问该网站并保持关注。由于部分站点可用性需人工验证，欢迎热心同学参与维护。

2.2 拥有科学上网能力：newBing、免费在线应用

如果你可以正常访问外网应用，但由于没有 OpenAI 账号，仍无法直接使用官网服务。但相对来说，在外网部署的基于 GPT API 的免费应用更多一些。例如：

  • 可以直接访问 new Bing，现在只要访问 IP 是其允许的地区，都能正常体验。
  • 访问第三方部署的在线应用网站。该网站收集了大量相关站点：lzwme/chatgpt-sites。需要注意的是，在使用第三方应用时应当注意安全，尽量避免填入个人敏感信息。

2.3 拥有 API KEY：自建在线应用

如果你有 API KEY，但科学上网并不是那么方便，除了使用免费的 ChatGPT 在线网站外，还可以基于开源程序搭建自己的站点：

  • 如果你有外网服务器，可以基于开源应用仓库直接在服务器上搭建
  • 也可以白嫖 vercel（需拥有自己的域名可绑定，免费的域名需科学上网）、netlify(可国内访问)等网站托管商的免费服务

支持可以一键部署到 vercel、netlify 服务的开源应用仓库参考：

  • chatgpt-demo A demo repo based on OpenAI API (gpt-3.5-turbo)
  • chatgpt-vercel 部署一个你自己的 ChatGPT 网站（免费）
  • chat-with-gpt About An open-source ChatGPT app with a voice
  • more…

提示：当前有一些第三方可以提供免费的外网代理服务，让你能够直接将服务部署在国内服务器，或者使用本地客户端类应用。但由于请求经过代理服务器时可能存在 API KEY 泄漏的风险，应注意仔细甄别此类服务。

2.4 有 VPN 和有 API KEY：自建在线应用、安装客户端应用

有此条件的同学已经不用为无法体验发愁了，但仍可能对使用便捷性、个性化使用等有一定的要求。有一些开源应用程序或客户端可能可以满足此类需求。

  • 白嫖 vercel、netlify 等免费服务搭建自己的个性化在线应用网站。具体可参考上一条。
  • 安装开源应用客户端或浏览器插件。由于这种客户端或插件是在你本地的，基本不用担心 KEY 泄漏的风险。
    • ChatBox 一个 OpenAI API 的跨平台桌面客户端
    • yetone/openai-translator 基于 ChatGPT API 的划词翻译浏览器插件和跨平台桌面端应用
    • BilibiliSummary A chrome extension helps you summary video on bilibili.
    • more…

3 OpenAI 入门学习

3.1 基于 OpenAI API 的应用开发

目前已经有很多开源应用基于 OpenAI API 实现，其核心逻辑是基于 completions API 发起 http 请求并处理响应结果。以下是示例代码：

const result = await fetch(`https://api.openai.com/v1/chat/completions`, {
  headers: {
    "Content-Type": "application/json",
    Authorization: `Bearer ${apiKey}`,
  },
  method: "POST",
  body: JSON.stringify({
    messages,
    model: "gpt-3.5-turbo",
    temperature: 0.5,
    max_tokens: 4096,
    stream: true,
  })
});

你可以编写程序构造对应的 body 参数，请求 completions API 并处理返回结果，进行相关应用程序的开发。
